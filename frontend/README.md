# 🎬 AI Movie Genius – Frontend

Suggests AI-powered movie recommendations based on user input using Claude API.

## 🛠️ Stack

- React + Bootstrap
- Axios for API integration

## 📦 Run

```bash
npm install
npm run dev
```
